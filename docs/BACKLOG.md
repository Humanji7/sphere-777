## B2B: Event Creatures (Промо-существа для мероприятий)

**Суть:** Предлагать организаторам событий кастомизированную "живую сферу" как интерактивный промо-элемент.

### Целевые сегменты

| Тип события | Пример кастомизации |
|-------------|---------------------|
| Tech-конференции | Цвета бренда, реакция на хэштег в Twitter |
| Музыкальные фестивали | Пульсация под BPM сцены, визуальная тема артиста |
| Корпоративы | Логотип компании в ядре, фирменные цвета |
| Арт-выставки | Интерактивный экспонат, реагирующий на посетителей |
| Стримы/онлайн | Виджет для OBS, реакция на донаты/подписки |
| Свадьбы/праздники | Персонализированные цвета, имена в анимации |

### Модель монетизации

1. **Разовая лицензия** — кастомная сборка под событие
2. **Подписка** — доступ к конструктору + хостинг
3. **White-label** — полная кастомизация под бренд агентства

### Технические требования

- [ ] Конфигуратор цветовой схемы (primary, secondary, accent)
- [ ] Загрузка лого/изображения в ядро сферы
- [ ] Интеграции: Twitter/X mentions, Twitch events, музыкальный BPM
- [ ] Embed-режим (iframe для сайтов)
- [ ] QR-код → мобильная версия для посетителей

### Каналы продаж

- Event-агентства
- Продакшн-студии
- Маркетинговые агентства
- Прямые обращения к организаторам (Timepad, Eventbrite)
- Стриминг-комьюнити (Twitch, YouTube)

### MVP для валидации

1. Сделать 2-3 демо-версии с разными темами
2. Написать 10 организаторам с предложением бесплатного пилота
3. Собрать фидбек, понять willingness to pay

### Уникальность предложения

- Не статичный баннер, а **живое существо с характером**
- Посетители могут взаимодействовать (touch, accelerometer)
- Запоминает и реагирует на аудиторию
- Эмоциональная связь > обычный промо-контент
