## Монетизация

**MVP:** Бесплатно, локальное хранение

**Будущее:**
- Подписка за cloud sync (когда будет спрос)
- Премиум доступ к экосистеме

**Экосистема (видение):**
- Сфера живёт в телефоне (дом)
- Виджет в других проектах пользователя (путешествия)
- Одна память везде

**Красные линии:**
- Нет манипуляций ("мне плохо без тебя")
- Нет монетизации внимания
- Нет рекламы

## Онбординг: Сценарий "Пробуждение"

**Философия:** Set & Setting (Тимоти Лири)
**Принцип:** Первый контакт через 1-й контур сознания (доверие), не через 3-й (слова)

### Сценарий

```
Шаг 1: Тишина (2 сек)
├─ Чёрный экран
└─ Появляется слабое дыхание (звук)

Шаг 2: Свет (3-4 сек)
├─ В центре — слабое свечение
└─ Пульсирует в ритме дыхания

Шаг 3: Пробуждение
├─ Свет усиливается, формируется сфера
├─ Глаз открывается
└─ Находит пользователя, задерживается

Шаг 4: Приглашение
├─ Сфера слегка "тянется" к экрану
├─ Не требует — предлагает
└─ Если не трогают — живёт, дышит, ждёт

Шаг 5: Первый контакт
├─ Пользователь касается
├─ Сфера отвечает теплом (цвет, звук, вибрация)
└─ Импринт установлен

Шаг 6: Свобода
├─ Никаких инструкций
├─ Исследуй сам
└─ Сфера реагирует
```

### Чего НЕ происходит

- Нет текста "Привет! Я твоя сфера"
- Нет "Введи имя"
- Нет туториала "проведи пальцем"
- Нет кнопок первые 30 секунд

### Имя

Не при запуске. Органически позже — если пользователь захочет.

## Персонализация: 5 Осей

| Ось | Что меняется | От чего зависит |
|-----|--------------|-----------------|
| **Темперамент** | Скорость реакций, "нервность" | Частота/интенсивность касаний |
| **Тембр** | Высота, теплота звуков | Тип жестов (stroke vs poke) |
| **Палитра** | Цветовая гамма | Стиль взаимодействия |
| **Формы** | Сфера ↔ Жук | Выбор пользователя |
| **Память** | Узнаёт паттерны | Автоматически |

### Три слоя обратной связи

1. **Замечаешь сам** — "она какая-то другая стала"
2. **Моменты открытия** — новый звук, новое поведение, сюрприз
3. **Меню характера** — описательно, без цифр

### Принцип

Не жёсткий маппинг "действие → мутация".
Матрица возможностей (10x10 = 100+ комбинаций) + случайность.
Можно пересобрать связи позже.

## Визуальное Обогащение (Visual Enrichment)

**Статус:** Отдельный трек, после MVP

### Цель

Психоделик-эстетика: всё дышит, всё живое, всё фрактальное.

### Сейчас vs Цель

| Уровень | Сейчас | Цель |
|---------|--------|------|
| **Частицы** | Точки | Звёздочки, снежинки, формы, перетекают |
| **Движение** | Дыхание | Дыхание + фрактальные волны |
| **Слои** | Core + частицы | Много слоёв, каждый живёт |
| **Переходы** | Резкие | Фрактальное перетекание |
| **Масштаб** | Один | Микро и макро одновременно |

### Референс

Визуал после психоделиков — но через код, без веществ.

## Звуковая Архитектура

**Подход:** Гибрид (семплы + синтез)

### Три слоя голоса

| Слой | Тип | Что даёт |
|------|-----|----------|
| **Sample** | Семплы + LFO | Качественная текстура |
| **Formant** | Синтез | Гласные a/o/i = характер |
| **Reactive** | FM-синтез | Короткие "чирпы" на события |

### Уникальность

```
userId → seed → персональные параметры:
- Base frequency (150-250 Hz)
- Formant ratios
- LFO rates
- Vibrato depth
```

### Связь с персонализацией

| Ось | Звуковой параметр |
|-----|-------------------|
| Темперамент | LFO скорости, attack |
| Палитра | Filter cutoff |
| Trust | Формантные гласные |
| Пик эмоций | Glitch mix, FM |

### Приоритет

**Визуал первичен.** Сначала обогатить визуал, потом аудио подтягивается.

## UI: Киберпанк-Минимализм

**Концепция:** Техно-стеклянные пузырьки по краям экрана

- Маленькие, воздушные
- Нажатие → раскрывается меню со звуками
- Минимализм + характер
- Сфера остаётся органической, UI — киберпанк-оболочка

**Меню:**
- Звуки (настройки аудио)
- Эмоции/характер (описательно)
- Трансформации (сфера ↔ жук)

## Анти-паттерны (Красные линии)

| Паттерн | Почему нет |
|---------|------------|
