# HANDOFF: LivingCore Эмоции — Слишком Красно

## Статус
- **Дата**: 2026-01-10 22:35
- **Коммит**: `041c55e` — LivingCore v2 (2 слоя, тёплая палитра)
- **Проблема**: Сфера выглядит "тревожной" даже в покое

---

## Проблема

LivingCore v2 использует **тёплую янтарную палитру** (`#FFAA66`, `#FFCC88`), но визуально это:
- Слишком **оранжево-красное** в нейтральном состоянии
- Создаёт ощущение **постоянной тревоги**
- Не хватает **спокойного/холодного** состояния как базы

---

## Идеи для решения

### A. Холодная база, тёплые эмоции
**Peace = нейтральный/холодный**, тепло появляется при взаимодействии:

```javascript
phaseColors: {
    peace: {
        inner: new THREE.Color(0x6688AA),  // Холодный серо-голубой
        pulse: new THREE.Color(0x88AABB)   // Прохладный
    },
    alert: {
        inner: new THREE.Color(0xBB8866),  // Тёплый коричневый
        pulse: new THREE.Color(0xCC9977)
    },
    trust: {
        inner: new THREE.Color(0xDDCC99),  // Тёплое золото
        pulse: new THREE.Color(0xEEDDBB)
    },
    bleeding: {
        inner: new THREE.Color(0xFF6644),  // Красно-оранжевый
        pulse: new THREE.Color(0xFF8866)
    }
}
```

### B. Монохромная серая база
**Peace = почти бесцветный**, цвет = эмоция:

```javascript
peace: {
    inner: new THREE.Color(0x888888),  // Нейтральный серый
    pulse: new THREE.Color(0x999999)
}
```

### C. Биолюминесценция — свет изнутри
Вместо цветных слоёв — **белое/голубоватое свечение** как у глубоководных:

```javascript
peace: {
    inner: new THREE.Color(0xCCDDFF),  // Холодный лунный свет
    pulse: new THREE.Color(0xDDEEFF)
}
```

### D. Прозрачность + particle glow
Убрать цвет из LivingCore → сделать его **почти невидимым**.
Цветовая эмоция передаётся через **particles** (уже реализовано).

---

## Рекомендация

**Вариант C (Биолюминесценция)** — холодный лунный базовый цвет:
- Peace = голубовато-белый
- При взаимодействии → тепло проступает
- Bleeding = красный прорывается через холод

Это создаст **контраст** и сделает тёплые эмоции **особенными**.
